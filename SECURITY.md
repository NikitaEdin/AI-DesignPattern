# Security Policy

## API Key Security
This research project requires API keys for various LLM services:
- Never commit API keys to the repository.
- Use environment variables or secure configuration files.
- Follow each provider's security best practices.

## Research Data Protection
- Ensure generated code samples contain no sensitive information.
- LLM API calls should not expose proprietary code patterns.
- Maintain data privacy standards throughout the research process.

## Academic Integrity
All security practices should align with institutional research guidelines and ethical standards for academic work.
